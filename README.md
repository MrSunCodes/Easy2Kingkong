# Easy2Kingkong
## 简介
本脚本由付导的录播组制作  当前版本为 V1.6 （20200201）
- 1.推送到Github以更好保存
- 2.优化连接速度
- 3.修复脸书的第三方登录
- 4.加快登陆二维码的出现
- 5.加快[未污染地址](https://www.kingkongapp.com/)连接速度
